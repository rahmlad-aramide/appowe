# AppOwe - Yoruba Proverbs App

## Overview

AppOwe (pronounced "Apowe") is a mobile application designed to preserve and promote Yoruba culture through a rich repository of proverbs. This app aims to address the gradual decline in traditional language usage among the younger generation, providing easy access to the wisdom embedded in Yoruba proverbs and their English equivalents.

## Features

- **Multilingual Support:** Offers content in Yoruba and English and also offers translation into 4 other languages: French, Spanish, Portuguese and German to cater for both native and non-native speakers.
- **Daily Proverb Updates:** Receive new proverbs daily to expand your knowledge and appreciation of Yoruba wisdom.
- **Social Sharing:** Share proverbs with friends and family on various social media platforms.
- **Favorites and History:** Save your favorite proverbs for quick access and view your history to revisit previously read proverbs.
- **Educational Tool:** An invaluable resource for academics, researchers, language learners, and anyone interested in the rich Yoruba culture.
- **Supports SDGs:** Contributes to the Sustainable Development Goal of Quality Education by enhancing linguistic and cultural education.
- **Economic Advantages:** Assists tourists and business professionals in understanding and communicating effectively in Yoruba-speaking regions.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- React Native CLI
- Android Studio or Xcode (for iOS)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/rahmlad-aramide/appowe.git
   cd appowe
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

   or if you use yarn:

   ```bash
   yarn install
   ```

3. **Running the application:**

   - **For iOS:**

     ```bash
     cd ios
     pod install
     cd ..
     react-native run-ios
     ```

   - **For Android:**

     ```bash
     react-native run-android
     ```

## Usage

Open the AppOwe application on your device to start exploring Yoruba proverbs. Use the drawer to switch between daily proverbs, favorites, history and other features available on the app.
